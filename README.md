# vscode-swan
百度智能小程序高亮与补全插件

## Features
 * `.swan` 文件高亮显示
 * API 智能提示
 * 代码片段补全

## Available settings
在开始使用vscode-swan前，您可以通过用户设置来开启或禁用API 的智能提示，默认为以下设置，即开启状态
```
{
    "vscode-swan.disableAPISuggestion": false
}
```

## Tips
API 智能提示功能仅在检测到工作目录为小程序目录（根目录下有 project.swan.json 文件）时生效
